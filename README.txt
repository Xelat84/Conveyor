src - папка с конвеером, готовым к подключению в качестве либы
examples-src - классы для компиляции примеров (используют конвеер из папки src)
test-src - Unit-тесты (там один класс и 14 тестов)
	Для запуска тестов надо скомпилировать и запустить класс RunTests.
	Для тестирования используется библиотека asunit v.3.
	Берётся отсюда: https://github.com/lukebayes/asunit
	(asunit v.4 не используется т.к. завязана на еще одно либу)
bin - скомпилированные версии примеров и тестов
